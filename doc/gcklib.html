1.1 Description ¶

GCKLIB is a gnulib-like C utility library for the GCK suite of package distributions and software. It provides reusable modules to simplify common C development tasks.

Previous: Description, Up: Overview   [Contents]

1.2 Quickstart ¶

This section details how to install and use GCKLIB. GCKLIB comes with the gcklib-tool, which can import, remove, and update modules added to a package. For more detailed information about any of the gcklib-tool functions, see the dedicated sections.

  • Install
  • gcklib-tool

1.2.1 Install ¶

GCKLIB is installed by adding it as a submodule. This is achieved with: 

git submodule add https://github.com/gck-org/gcklib

It is important to note that if you want to contribute to a project that uses GCKLIB and want to use a new module, you must clone it recursively: 

git clone --recursive https://github.com/gck-org/gcklib

1.2.2 gcklib-tool ¶

gcklib-tool is a shell script used to import, remove, and update modules added to a package. To use gcklib-tool, first cd into the root of the gcklib directory.

  • import
  • remove
  • update

1.2.2.1 import ¶

./gcklib-tool --import module

Imports the named module into the current package.

1.2.2.2 remove ¶

./gcklib-tool --remove module

Removes the named module from the current package.

1.2.2.3 update ¶

./gcklib-tool --update [module]

Updates all modules in the current package, or the specified one if provided.

2 API Reference ¶

This section describes selected variables, functions, and macros exported by GCKLIB.

int: Variable gck_example_count ¶

This variable tracks the number of example objects managed by GCKLIB.

Function: void gck_example_init (void) ¶

Initializes the GCK example subsystem.

Function: void gck_example_free (void) ¶

Frees resources allocated by the GCK example subsystem.
